# Collect the Gifts

A fun pygame where you help Santa collect gifts into his bag!

## Game Description

Help Santa collect gifts into his Christmas bag! You have 30 seconds to collect as many gifts as you can. 

### How to Play

1. **Goal**: Drag gifts into Santa's green bag at the bottom of the screen
2. **Controls**: 
   - Click and hold on a gift to drag it
   - Drop it into the bag to score a point
3. **Time Limit**: You have 30 seconds to get the highest score possible
4. **Respawning**: When all 10 gifts are collected, 10 new gifts spawn automatically
5. **Movement**: Gifts float around the screen slowly - catch them before time runs out!

## Installation

1. Install Python 3.7 or higher
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Game

```bash
python game.py
```

## Game Features

- 10 gifts spawn at the start
- 30-second timer
- Gifts float around the screen
- Drag and drop mechanics
- Score tracking
- Auto-respawn when all gifts are collected
- Game over screen with final score
